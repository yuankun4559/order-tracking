# README

## 环境说明

### 一，dev(开发环境)

`````bash
    1，打包命令：
        npm install 或 yarn
        npm run build:dev
    2，访问地址  dev.order-tracking-agreement.ruigushop.com
```

### 二，test(测试环境)

````bash
     1，打包命令：
        npm install 或 yarn
        npm run  build:test
     2，访问地址  test.order-tracking-agreement.ruigushop.com
```

### 二，uat(uat 环境)

````bash
     1，打包命令：
        npm install 或 yarn
        npm run  build:uat
     2，访问地址  uat.order-tracking-agreement.ruigushop.com
```

### 三，master(正式环境)

````bash
     1，打包命令：
        npm install 或 yarn
        npm run  build:prod
    2，访问地址 https://order-tracking-agreement.ruigushop.com
`````
